Advanced concordion expressions

OGNL (Object Graph Notation Language) is the expressions used to reference properties, methods and variables inside of the concordion test.
To enable using of OGNL expressions annotate your test fixture with @FullOGNL.
Concordion supports complex expressions, but they should be used with caution and follow KISS principle as this is a testing code.
Listed below expressions are valid with concordion.
